QUESTION

Can **Launch** be used to **host a custom Marketplace app**?

_Keywords: host marketplace app on Launch, custom app hosting, Developer Hub app hosting, Contentstack app, marketplace app deploy, app hosting._

ANSWER

- **Yes.** Launch can host a custom Marketplace app.
- Use **Developer Hub → App Hosting** to deploy/host it. Doc: https://www.contentstack.com/docs/developers/developer-hub/app-hosting

## Generalized guidance

- **"Can Launch host my custom Marketplace / Contentstack app?"** → Yes — host it via **Developer Hub App Hosting** (the documented path for app hosting on Launch).
