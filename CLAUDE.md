# Agent instructions for this repository

This repo holds **Markdown knowledge-base entries** distilled from Launch support conversations.

## How to create or update an entry

1. Paste or attach the **raw support thread** (Slack export, screenshot text, forwarded email chain).
2. Produce (or refine) an MD file using the **support pattern** imported below.

## Rules

The authoritative extraction rules live in `.cursor/rules/support-pattern.mdc` and are imported here so Claude follows the same pattern as Cursor:
